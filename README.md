练习总结
    1. 练习了 Vant2 的使用
        1-1. 主要练习了 Cell 组件，用于设置单行样式，注意使用时可以不设置高度，其可以自适应
                Cell 组件可以内部使用内置的slot，如#title --- 标题    #label --- 注释
        1-2. 练习了 PullRefresh 组件，用于下拉刷新
             练习了 List 组件，用于下拉加载
        1-3. 练习了 Tabbar 组件，用于设置底部栏
             练习了 NavBar 组件，用于设置顶部栏
    2. 练习了后台数据的处理
             练习了 axios 的常用封装方法
             练习了下拉刷新和下拉加载 两者的处理数据方法
